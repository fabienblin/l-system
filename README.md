# L-System

## Description
This project is a L-System tool that allows to implement any grammar as defined by this wikipedia article : https://en.wikipedia.org/wiki/L-system
You can also Ask ChatGPT after providing an example like the Fractal Tree.

## How-To
To implement a grammar you will need to implement a lsystem.DisplayableGrammar like in the examples from the */grammar* directory.

## Build Requirements
If you want to compile this project, you will also need to install Fyne v2 as described here : https://docs.fyne.io/started/quick/
